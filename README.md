# crypti-sandbox

Crypti Sandbox that run sandboxed Nodejs.

## Requires

    * Crypti 0.5.0

## Installation

## Requires

  * Crypti 0.5.0

## Installation

```sh
npm install crypti-sandbox
```

## Debugging

Run Crypti with next env variable:


```sh
DEBUG=1 node app.js
```
